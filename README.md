# 滑板人生 - 第2关：热带夏日

🛹 一个热带风格的 HTML5 3D 滑板闯关游戏

## 游戏玩法

- **目标**：控制滑板小人躲避障碍物，到达终点
- **操作**：← → 方向键 或 A/D 键控制移动
- **得分**：躲避障碍物 + 跑酷距离

## 视觉主题

- **热带夏日风格**
  - 主色调：橙色(#FF8C00)、金色(#FFD700)、青绿色(#00CED1)
  - 背景：热带夕阳渐变天空
  - 装饰：棕榈树、太阳、草帽
  - 地面：沙滩颜色(#F4A460)

## 平台支持

- ⏳ **Web 端**: (暂未部署，直接打开 `index.html` 即可运行)
- 🔜 **Android**: 计划中

## 技术

- Three.js 3D 渲染
- Canvas 2D / WebGL
- 响应式设计（适配手机和桌面）

## 运行

直接在浏览器中打开 `index.html` 即可游玩。
