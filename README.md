# 滑板人生 - 第1关：粉色浪漫

🛹 一个可爱风格的 HTML5 滑板闯关游戏

## 游戏玩法

- **目标**：控制滑板小人躲避障碍物，到达终点
- **操作**：← → 方向键 或 A/D 键控制移动
- **限时**：60 秒
- **得分**：躲避障碍物 + 通关奖励

## 视觉主题

- 粉色浪漫风格（#FF69B4, #FFB6C1, #FFC0CB）
- 棉花糖云朵、爱心、星星装饰
- 粉色渐变天空

## 技术

- 纯 HTML/CSS/JavaScript
- Canvas 2D 渲染
- 响应式设计（适配手机和桌面）

## 运行

直接在浏览器中打开 `index.html` 即可游玩。
